## Q2 Problem: Tribonacci Number

Given an integer `n`, return the **n-th Tribonacci number**.

The Tribonacci sequence is defined as:

- `T(0) = 0`
- `T(1) = 1`
- `T(2) = 1`
- `T(n) = T(n-1) + T(n-2) + T(n-3)` for `n ≥ 3`

---

## Example 1

```text id="z4m7kx"
Input: n = 4
Output: 4
```

---

## Example 2

```text id="p9c2vw"
Input: n = 5
Output: 7
```

---

## Example 3

```text id="x1n8rs"
Input: n = 0
Output: 0
```

---

### 🔴 Brute Force Approach (Recursion)

```cpp id="w7k3pt"
#include<iostream>
using namespace std;

// BRUTE FORCE: Recursive (O(3^n))
int tibo(int n){

    if(n == 0) return 0;
    if(n == 1 || n == 2) return 1;

    return tibo(n-1) + tibo(n-2) + tibo(n-3);
}

int main(){

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    cout << tibo(n);
}
```

---

### 🟢 Optimal Approach (Iterative DP)

```cpp id="k6r9yu"
#include<iostream>
using namespace std;

// OPTIMAL: Iterative (O(n), O(1) space)
int tibo(int n){

    if(n == 0) return 0;
    if(n == 1 || n == 2) return 1;

    int prev3 = 0; // T(0)
    int prev2 = 1; // T(1)
    int prev1 = 1; // T(2)

    for(int i = 3; i <= n; i++){
        int curr = prev1 + prev2 + prev3;
        prev3 = prev2;
        prev2 = prev1;
        prev1 = curr;
    }

    return prev1;
}

int main(){

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    cout << tibo(n);
}
```
