## Q1 Problem: Fibonacci Number

Given an integer `n`, return the **n-th Fibonacci number**.

The Fibonacci sequence is defined as:

- `F(0) = 0`
- `F(1) = 1`
- `F(n) = F(n-1) + F(n-2)` for `n ≥ 2`

---

## Example 1

```text
Input: n = 2
Output: 1
```

---

## Example 2

```text
Input: n = 5
Output: 5
```

---

## Example 3

```text
Input: n = 10
Output: 55
```

---

### 🔴 Brute Force Approach (Recursion)

```cpp
#include<iostream>
using namespace std;

// BRUTE FORCE: Recursive (O(2^n))
int fibo(int n){

    if(n == 0) return 0;
    if(n == 1) return 1;

    return fibo(n-1) + fibo(n-2);
}

int main(){

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    cout << fibo(n);
}
```

---

### 🟢 Optimal Approach (Iterative DP)

```cpp
#include<iostream>
#include<sstream>
#include<vector>
using namespace std;

// OPTIMAL: Iterative (O(n), O(1) space)
int fibo(int n){

    if(n == 0) return 0;
    if(n == 1) return 1;

    int prev1 = 1; // F(1)
    int prev2 = 0; // F(0)

    for(int i = 2; i <= n; i++){
        int curr = prev2 + prev1; // F(i)
        prev2 = prev1;
        prev1 = curr;
    }

    return prev1;
}

int main(){

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    cout << fibo(n);
}
```
